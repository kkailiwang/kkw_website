    Pear Program

    See our 2021 published article here

    Pear Program is a Stanford-funded educational web application that allows students to pair-program effectively. It could be thought of as Google Docs for computer science education. Unlike other collaborative IDEs such as VSCode LiveShare (which is mostly for software developers), PearProgram integrates educational research into the tool to foster better coding practice. For example, there are question and comment features that allow students to flag code when they do not follow along. Each student has one of two possible roles: Pilot and Co-Pilot. Thus, they may not edit code at the same time, to ensure that they are learning effectively and work at the same pace.

    I built all of the front end (React, Javascript, HTML/CSS, data stream network) and some of the backend (Node, serverless architecture, AWS Lambda, AWS DynamoDB, AWS Amplify). My team also consists of 4 Stanford PhD students in the Graduate School of Education. Our sponsor and PI is Chris Piech, professor of the Computer Science Department.

    PearProgram has already been used in three separate course offerings since April 2020: CS106A, the introductory CS class at Stanford, Stanford Code in Place, a free global offering of CS106A launched in the wake of COVID, and also CSBridge, the international introductory program that was virtual this year.

    POPS Project

POPS is based out of Piedmont, a resourceful area of Oakland in California, and provides an action hub of a website that raises money to support less privileged areas in Oakland and strive for racial equality in the education system.
